Day Trip Planner

Software Design for Handheld Devices

Spring 2020


Android Application to search for restaurants, metro stops, and activities near a user-inputted location. On the main screen, the user can specify location, restaurant category, attraction category, and number of results for all categories. The app makes a call to the Yelp Business Search API and WMATA API. The Map view shows the results that are returned by the API calls. There are also more detailed information about each business shown in a list format.
